Sobre mi

Desde el año 2020 comencé a estudiar todo lo concerniente al Desarrollo Web Front End. He realizado diversos cursos sobre las tecnologías más demandadas en la actualidad, dentro de las cuales destacan: HTML, CSS y JavaScript.

Actualmente, me desempeño como Tutor del curso de Desarrollo Web de Coder House, lo cual, además de mantenerme activo y practicando, también me permite ayudar a los alumnos en su proceso de aprendizaje, corrigiendo sus desafíos y aconsejándolos lo más posible para que puedan cumplir su objetivo exitosamente.

¿Qué hago?

Entre las tecnologías con las que más he trabajado están: HTML 5, CSS3, JavaScript, jQuery, Bootstrap y SASS para todo lo relacionado con el desarrollo; GIT para control de versiones, y Github como repositorio de código on-line.
